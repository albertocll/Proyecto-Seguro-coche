✅ Comprobador de Seguro – Prototypes en JavaScript
Este proyecto fue creado con el objetivo de practicar y aplicar el uso de prototypes en JavaScript mediante un ejercicio funcional: calcular el costo de distintos tipos de seguros. Está orientado a fortalecer conceptos clave como herencia prototípica, creación de objetos y separación de lógica.

📋 Características
Calcula el costo de un seguro según marca, año y tipo.

Usa prototypes para estructurar y reutilizar la lógica.

Interfaz simple directamente en el navegador.

Desarrollado íntegramente con JavaScript puro (Vanilla JS).

⚙️ Tecnologías utilizadas
HTML5 – Estructura de la página.

CSS3 – Estilos personalizados.

Tailwind CSS – Utilizado como base de diseño.

JavaScript (Vanilla) – Lógica basada en prototypes.

🚀 Cómo usar
Clona este repositorio o descarga los archivos.

Abre el archivo index.html en tu navegador.

Usa el formulario para simular un cálculo de seguro.
